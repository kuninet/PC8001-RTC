= PC-8001 外付け RTC-7301DG ボード

== 概要
* PC-8001の拡張バスに接続するリアルタイムクロックIC RTC-7301DGボードです。

== I/Oアドレス
* 80h〜F0hのあいだで10h単位にジャンパで設定できます。
** RTC-7301DGはA0〜A3までの4本のアドレス線を使用するため、ジャンパで設定したベースI/Oアドレスが80hの場合 80h〜8Fhまでを使用します。
